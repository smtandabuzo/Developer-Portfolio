#2 - Installing Template

After unzip Template Files, you will found HTML files and all other files. You can view this template in any web browser from your desktop computer. Because the files are written in HTML, you do not need an internet connection in order to display or edit the template. Only contact form need server or localhost in order to work as its using php for sending form data directly into your mail. Login to your cPanel, locate to the domain folder you wish to upload your files to on your server then transfer all your files to server.

#3 - HTML

This Template has a Responsive layout with a variation of column layouts depending on device. This Template uses Twitter Bootstrap grid system(12 columns) which will help streamline any customization you wish to make. If you're unfamiliar with Twitter Bootstrap grid system, be sure to go to Twitter Bootstrap documentation, and you will see the Grid system with divs.

- Intro

For intro text below given Html is written, you can easily find that and replace with your own content.

<h1 class="cd-headline rotate-1 is-full-width"><span>John Doe <small class="cd-words-wrapper"><b class="is-visible">Front End Developer</b><b>WordPress Expert</b><b>Entrepreneur</b></small></h1>
- About

In this section, you can write anything about yourself e.g. your name, mobile number, website url, email etc. You can also add a image of yourself here alongwith the skills.

- Services

This section is to let you know your client, what services you provide. You can add as many services as you want.

- Portfolio

Show some of your best work to the world. you can add as many as you want.

- Resume

What is your qualifications, how much expericence do you have, you can show everythign here.

- Testimonial

What your client say about your work, will be visible here. Each <li> is used for each testimonials. You can add as many as you want.

- Blog

This section is created to show your latest blog post.

- Contact

This section will help you to add your contact details, social websites links also the map. So that your client can reach to you easily.

#4 - CSS

This template uses one main CSS file (include/css/style.css) in this Template which you are free to modify depending on the customization you require. There are also few additional style sheets that you can do changes if needed.

#5 - Javascript

You can do changes in Custom.js file (include/js/custom.js). There are also few additional javascript files that We do not recommend changing until you know the language very well.

#6 - Contact Form

This is the working form for you. No need to add anythign, just upload and it will start working. Only you have to change the destination email, so that when a user submit the form, you can directly get data into your email. to change the currect emai address, you have to open mail.php available on root and change the value given below

$to = "service@slicemytheme.com";

currently we have used service@slicemytheme.com as desitnation mail but you can change it with your email and then upload the file on server.
